# Event Map App

This project is a Next.js application that displays events on a map. Events are scraped from the Meetups and EventBrite APIs, and users can create their own events.

## Features
- Display events on an interactive map.
- Integrate with Meetups and EventBrite APIs.
- Allow users to create and manage their own events.

## Tech Stack
- **Next.js**: Framework for server-rendered React applications.
- **TypeScript**: For type safety and better developer experience.
- **Tailwind CSS**: For styling.

## Getting Started
1. Clone the repository.
2. Install dependencies with `npm install`.
3. Run the development server with `npm run dev`.
